Instructions for drawing the cape

  • The instructions are divided into steps.

  • 1. Decide on the size of the cape you want.

  • TNTClient supports the following cape sizes: 64x32 (standard size), 128x64, 256x128 (recommended resolution), 512x256 and 1024x512. The larger the cape size, the sharper the image, but it also takes up more space in the video memory.

  • 2. Downloading the required template.
  • There are 2 formats:
    • psd - For those who have Photoshop.
    • png - Can be edited using any graphics editor (e.g. Paint)
  • Templates:
    • 64x32: psd, png
    • 128x64: psd, png
    • 256x128: psd, png, psd (Example), png (Example)
    • 512x256: psd, png

  • 3. Draw what you want to see.
  • The most difficult stage, but I believe you can do it.
    • You don't have to draw, you can insert any picture from the internet.

  • 4. Check on this website to see how it will look on you and if there are any mistakes.
